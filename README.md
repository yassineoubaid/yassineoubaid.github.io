# Yassine Oubaid - Personal Website

Static personal academic website for Yassine Oubaid.

## Project structure
- `index.html` - homepage
- `styles.css` - styling
- `script.js` - small UI interactions
- `CV_yassine_OUBAID_Postdoc/` - CV PDF, LaTeX source, and portrait photo
- `.github/workflows/deploy.yml` - automatic GitHub Pages deployment

## Publish with GitHub Pages
1. Create a GitHub repository named `yassineoubaid.github.io`.
2. Push this project to the `main` branch of that repository.
3. In GitHub, open `Settings > Pages`.
4. Set `Source` to `GitHub Actions`.
5. The included workflow will deploy the site automatically on every push to `main`.

Once published, the website URL will be:

`https://yassineoubaid.github.io/`

## Local preview
Open `index.html` in a browser.
